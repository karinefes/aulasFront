Alinhamento de elementos e conteúdo usando flexbox

flexbox é um mecanismo do CSS3 que permite aplicar facilmente alinhamento e distribuição de elementos conteúdos no HTML.

Na prática, é uma técnica diferente para solucionar problemas e minimizar o grande uso de paddings, margins, floats, positions na resoluçãode dificuldades na costrução de layouts.

As propriedades do flexbox são os seguintes:

  • Flex-direction: Define qual direção o container deseja empilhar os itens (column, column-reverse, row, row-reverse).
  • Flex-wrap: Epecifica se os itens flexiveis devem ser encapsulados ou não (flex-wrap: wrap, nowrap, wrap-reverse).
  • Flex-flow:Propiedades abreviada para definir flex-direction e flex-wrap (flex-flow: row wrap).
  • Justify-content: Alinhamento de itens (justify-content: center, flex-start, flex-end, space-around, space-between).
  • Align-itens: Propriedade é usada para alinhar os itens flexiveis.
  • align-content: Propriedade é usada para alinhar as linhas flexíveis.
